# Blog

## Theme Features:

- ✅ Dark and light color mode
- ✅ Hero section with bio
- ✅ Pagination support
- ✅ Post tags support
- ✅ View transitions
- ✅ Tailwind CSS
- ✅ Mobile-first responsive layout
- ✅ SEO-friendly with canonical URLs and OpenGraph data
- ✅ Sitemap support
- ✅ RSS Feed support
- ✅ Markdown & MDX support
